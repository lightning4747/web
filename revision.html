HTML Basics

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page.

Basic HTML Document Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
</body>
</html>

Common HTML Elements

Example:

HTML Code

<h1>Main Heading</h1>
<h2>Subheading</h2>
<p>This is a paragraph with <strong>bold text</strong> and <em>italic text</em>.</p>
<a href="#">This is a link</a>
<img src="image.jpg" alt="Description">
<ul>
    <li>List item 1</li>
    <li>List item 2</li>
</ul>

CSS Basics

CSS (Cascading Style Sheets) is used to style and layout web pages.

CSS Syntax

selector {
    property: value;
}

CSS Selectors

Example:

CSS Code

/* Element selector */
p {
    color: blue;
}

/* Class selector */
.highlight {
    background-color: yellow;
}

/* ID selector */
#main-title {
    font-size: 2rem;
}

/* Grouping selectors */
h1, h2, h3 {
    font-family: Arial, sans-serif;
}

CSS Box Model

Every element in web design is a rectangular box. The CSS box model describes this box with content, padding, border, and margin.

Box Model Interactive Demo

Box Model Example:

CSS Code

.box {
    width: 200px;
    height: 100px;
    padding: 20px;
    border: 5px solid #6a11cb;
    margin: 30px;
    background-color: #f0f5ff;
}

Result

Content Area

Total width: 200px (width) + 40px (padding) + 10px (border) = 250px
Total height: 100px (height) + 40px (padding) + 10px (border) = 150px

CSS Layout

CSS provides several layout techniques for positioning elements on a web page.

Flexbox

Flexbox is a one-dimensional layout method for arranging items in rows or columns.

Flexbox Example:

CSS Code

.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
    gap: 10px;
}

Responsive Design

Responsive web design makes your web page look good on all devices.

Media Queries

Media queries allow you to apply CSS styles depending on device characteristics.

Media Query Example:
/* Default styles for mobile */
.container {
    padding: 10px;
}

/* Styles for tablets */
@media (min-width: 768px) {
    .container {
        padding: 20px;
    }
}

/* Styles for desktops */
@media (min-width: 1024px) {
    .container {
        padding: 40px;
        max-width: 1200px;
        margin: 0 auto;
    }
}

Viewport Meta Tag

The viewport meta tag controls the layout on mobile browsers.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

CSS Animations

CSS allows you to create animations without JavaScript.

Transitions

CSS transitions allow you to change property values smoothly over a given duration.

Transition Example:

CSS Code

.button {
    background-color: #6a11cb;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    transition: all 0.3s ease;
}

.button:hover {
    background-color: #2575fc;
    transform: scale(1.05);
}

Result

Keyframe Animations

CSS keyframe animations allow you to create more complex animations.

Keyframe Animation Example:

CSS Code

@keyframes bounce {
    0%, 100% {
        transform: translateY(0);
    }
    50% {
        transform: translateY(-20px);
    }
}

.animated-box {
    width: 50px;
    height: 50px;
    background-color: #6a11cb;
    animation: bounce 2s infinite;
}
